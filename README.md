# WokAI & Horizonte Global | Panel de Control

Panel de control de gestión: calculadora de tarifas, objetivos vs realidad, cartera de proyectos, planilla mensual, CRM y costos & contribución marginal.

- Aplicación de un solo archivo (`index.html`), sin dependencias externas.
- Las horas llegan de Toggl Track y el tipo de cambio del BNA mediante rutinas diarias que actualizan y republican este archivo.
- Los datos de gestión (cartera, registros, estimados) viven en el `localStorage` del navegador; usar «Descargar respaldo» / «Restaurar respaldo» para moverlos entre dispositivos.
